For this project, I created a personal "About Me" webpage. I created this page by adding a navigation bar, headers, paragraph, clickable images, unordered lists and formatting. These elements include HTML and CSS. I will be highlighting a few elements in my code to show you how to add these so that you can make you own site too.


Navigation Bar

To make the navigation bar, I made a CSS file called navbar.css. The code in .navbar { programs the color of the navigation bar and the way that the buttons are aligned. I wanted the navigation bar to be grey because of the color scheme of blue and grey. Because I only have two buttons, I wanted them to be centered.
The code in .navbar a{ programs the buttons. The display: inline-block; makes sure that if the buttons fit on one line next to eachother, they would appear on the same line. I used this because I wanted the navigation bar to be one horizontal line rather than a vertical bar. The code color: black; makes the text color of the buttons, like "About Me," black. I chose black because I wanted to make the text visible on the white button background.
.navbar a:hover{ controls how the button looks when you hover your mouse over it. In this case, since the code is background-color: #e6e6e6;, the button turn grey when you hover over it.

When making headers, you use the header tag. There are 6 sizes with the first being the biggest, and the sixth being the smallest. In this photo, I used the second biggest header, the <h2> tag. I chose to use <h2> because my <h1> was the top header abve the navigation bar that read "Bella Kwong."

Clickable Images

Click the image to see an enlargement of the screenshot

This is an example of a clickable image

One specific feature I learned how to do and will remember is making a clickable image. I will remember how to make these because of how cool and useful they can be. When making clickable images, you add a link tag and an image tag together. A link tag looks like <a href="add link here" target="_blank">What you want the link to be called</a>. This makes text clickable so that it leads to another page. An image tag looks like <img src="image URL here">. <a href> means hyperlink reference. target="_blank" opens up the link in a new tab rather than that same one. If you want a clickable image that leads to another page, you put the image tag in the part of the link tag that says "What you want the link to be called." It should look like this: <a href="Add link here" target="_blank"><img src="image URL here"></a>. In the photo to the left, you can see that I made it so that when you click on an image, which are the sketches under Hobbies and Interests, it leads you to my art Instagram account. I decided to make it linked because I wanted to emphasize how art is my hobby and if yu want to see more, you can.


Unordered Lists

When making an unordered list, you have to have a set of unordered list tags, which are <ul>/</ul>. Inside of that you have to have list item tags, which are <li>/<li>. Inside the list item tags, you have to write what you want in bullet points. For example, for the "What I've Learned" section, I used the code <li>Headers<li> to get "Headers" in a bullet pointed list. I used bullet points rather than paragraphs because I wanted this section to be easy to read and easily able to add more items as I learn more in HTML and CSS.

Formatting with CSS

When formatting elements of your webpage, you have to use CSS. I created a new CSS file called index.css. Anything in the body {, affects the entire page. An example of what you can use this for is for changing the font with the code font-family: Open Sans;. I decided to make the font on my page Open Sans because that's one of my favorite fonts.
I wanted to have my "Hobbies and Interests" and "What I've Learned" on the right and "About Me" on the left. I wanted to format my site this way to make the webpage easy to navigate. I have the code .leftside{ and .rightside{ to divide the page in half and into sections. All the things I wanted on the left and right side are wrapped in their according divs. Another example of this formatting is how on this page all my text and explanations are on the right with the photos on the left.
