Generative Art How To

For my Generative Art project, I made a program that makes fish appear when you click in the ocean. It uses different types of Javascript functions, loops, and conditionals. This webpage will show you how I coded it.

The Ocean

In order to make the ocean, I used a for loop in the draw function. The draw function looks like draw = function(){. Anything in it will constantly redraw itself over and over again every frame. Inside it, I used a 'for loop', which looks like for (var... I used this because the ocean is actaully a blue rectangle. The loop is a chain of light blue circles that are constanly moving to the right. I could have coded a set of indivisual circles to constantly move to the right, but a loop is more efficient.

for (var i = 0; i < width; i += 80) controls where on the y-axis the ovals are. The first part, var i = 0;, is executed before the loop begins. The second statement, i < width, is the condition for running the third statement. So in order for i += 80, the variable i has to be less than the width. The last statement is executed everytime after the loop, as in the code under/inside it was executed. For example, after line 24, the loop will start from for (var j = startPlace; j < height; j += 80){. This controls where on the x-axis the ovals move. The computer runs this since j controls the y-axis. The code underneath it specifically draws the shape (ellipse) and colors is (fill). The code noStroke(); is needed because without it, it looks like this:

startPlace++ is an increment of my variable startPlace. This is what makes the "waves" actually move. The code below...
if (startPlace >= -300){

startPlace = -600;
}
...programs the ovals so that when they go past a certain point (-300px), the chain will move back to the left to its orignal starting point off screen (-600px).

Click for Fish

In order to make the fish show up when it was clicked, I coded this array, fishStay. This was above the draw function. You can see it in the first image to the right. Without this array, the background will constantly draw itself over the fish and the fish will appear to be flashing. This is also why the variable is called "fishStay."

The second right photo shows the for loop made for it. k = 0; is initiated before the loop starts. Once k < fishStay.length is true, then k will increment by 1 every time the loop is repeated (k++).


Not in the Sky

This block of code shown on the right is a conditional, or an if statement. I programmed this so that when you click above 300px on the y-axis, a fish will not be drawn. I did this by using mouseY, which uses the y-axis from the mouse. The code fishStay.push({x:mouseX + 130, y:mouseY - 50}); makes it so that when the mouse is clicked (.push), a fish is drawn. The numbers 130 and -50 make the fish appear exactly where the mouse cursor is. Without it, it looks like the gif to the right.
